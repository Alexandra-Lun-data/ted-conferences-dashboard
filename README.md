
# TED Conferences Dashboard

## Project Description
This project was developed for a company preparing its first TED-style conference.

The goal was to analyze historical TED conference data and build an interactive dashboard to support event planning decisions.

## Business Goal
The client needed to understand:
- which topics are most popular,
- how audiences react to talks,
- how many speakers are typically involved,
- what talk duration is optimal,
- which speakers generate the highest engagement.

## Tools
- Yandex DataLens
- SQL (data preparation)
- Data Visualization
- Dashboard Design

## Dashboard Link
https://datalens.yandex/70bwaqqqoz36r

## Dashboard Structure

### 1. Overview
- total number of conferences
- total number of talks
- number of speakers
- number of tags

### 2. Conference Analysis
- average talk duration
- maximum views
- total applause
- number of speakers

### 3. Talk Analysis
- top 10 funniest talks (by laughter)
- top 20 most popular tags

### 4. Speaker Analysis
- occupations with highest audience engagement (applause)

### 5. Detailed View
- full information about each talk

## Filters
- country
- conference name
- tag
- date

## Business Value
The dashboard helps:
- select topics for future conferences
- optimize event duration and structure
- identify high-performing speaker profiles
- improve audience engagement strategy
