# Data Description

## events
- conf_id — conference ID  
- event_name — conference name  
- country — country  

## speakers
- author_id — speaker ID  
- speaker_name — name  
- speaker_occupation — occupation  
- speaker_description — description  

## talks
- talk_id — talk ID  
- title — talk title  
- description — description  
- film_date — date  
- duration — duration  
- views_count — views  
- main_tag — main topic  
- speaker_id — speaker ID  
- laughter_count — laughter events  
- applause_count — applause events  
- language_sp — language  
- event_id — conference ID
